# Career Compass: Career Intelligence Platform 🧭

> **Development Status:** Currently optimizing API data-fetching routines and implementing SRE-focused health monitoring. Live deployment currently taken down. 🚀

## 📌 Project Overview
Career Compass is a data-centric career guidance platform developed to assist students in navigating the tech job market. It bridges the gap between raw job data and actionable insights using the MERN stack and modular REST APIs.

## 🛠 Technical Stack
* **Frontend:** React.js, Chart.js (Analytical Visualizations), CSS3
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Data Integration:** RESTful APIs for real-time market trends

## 🏗 Key Engineering Features
* **Analytical Dashboards:** Leveraged **Chart.js** to visualize salary distributions and skill demand across different Indian tech hubs (Bangalore, Pune, Delhi).
* **Reliability & SRE:** In alignment with **Site Reliability Engineering (SRE)** principles, I implemented a `/api/health` monitoring endpoint to track server uptime and API latency trends.
* **Agile Methodology:** The project followed an Agile-style SDLC, moving from requirements gathering to functional documentation and modular development.
* **Data Integrity:** Implemented server-side validation using Joi/Zod to ensure that user profile data and skill-matching inputs are clean and structured.



## 📈 SRE & Performance Focus
* **Latency Monitoring:** Integrated middleware to log response times, ensuring that the dashboard remains responsive even when fetching large datasets.
* **Error Handling:** Developed a centralized global error-handling mechanism to capture backend exceptions before they reach the retail user.

## 🛠 Setup & Installation
1.  **Clone:** `git clone https://github.com/Paras0544/Career-Compass.git`
2.  **Dependencies:** Run `npm install`.
3.  **Launch:** Execute `npm start` to view the application in the browser.
